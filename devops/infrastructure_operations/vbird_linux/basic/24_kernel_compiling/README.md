## 24.1 编译前的任务：认识内核与取得内核源代码

### 24.1.1 什么是内核（Kernel）

- Kernel: 管理硬件的软件，包含驱动硬件的检测程序与驱动模块
  - 内核文件通常为 `/boot/vmlinuz-xxx`
- 内核模块（Kernel module）的用途：动态载入模块，而不需要变动内核
  - 内核模块通常在 `/lib/modules/$(uname -r)/kernel/` 中
- 自制内核 - 内核编译
- 关于驱动程序 - 是厂商的责任还是内核的责任？
  - 是硬件尝试的责任
