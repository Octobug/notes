## 动态语言简介

能够在运行时修改自身程序结构的语言，就属于动态语言。比如：

- 在运行时给某个类增加成员函数及成员变量；
- 在运行时改变某个类的父类；
- 在运行时创建出某个函数；
- ...

### 脚本语言 vs. 动态语言

两者有很大的交集但并不等价。比如：

- C# 在 4.0 之后就可以算是动态语言了，但它不能算是脚本语言；
- 很多 Shell 脚本语言不能算是动态语言。
