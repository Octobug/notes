## 递归三要素

1. **确定递归函数的参数和返回值**：确定哪些参数是递归的过程中需要处理的，并且明确每次递归的返回值是什么；
2. **确定终止条件**；
3. **确定单层递归的逻辑**：确定每一层递归需要处理的信息，这里会重复调用自己来实现递归的过程。
